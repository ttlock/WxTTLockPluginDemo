# 通通锁蓝牙模块通信插件接口

## 方法  
```
    function setShowLog(open: boolean, callback?: (...args: any) => any): void
```  

### 功能描述   
 开启/关闭用户调试日志，允许用户自定义输出方法  
 方法未被调用时，默认不输出操作日志  

### 版本支持   
 在线版最低支持版本： **1.4.1**   
 离线版最低支持版本： **1.0.0**   

### 参数说明 
 |PARAMS    |TYPE                                  |REQUIRED      |DESCRIPTION|
 |----------|--------------------------------------|--------------|-----------|
 |open      |boolean                               |Y             |是否需要输出调试日志，默认false|
 |callback  |(...args: any) => void                |Y             |日志回调方法, openLog为false时不生效|  

### 返回值  
 void
